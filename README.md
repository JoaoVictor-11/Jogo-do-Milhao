Projeto de Extensão - 2025.2
Introdução

Este projeto de extensão tem como objetivo desenvolver um jogo interativo de perguntas e respostas, inspirado no clássico “Show do Milhão”. A proposta visa proporcionar aos alunos uma experiência prática e divertida na aplicação de conceitos de lógica de programação, estrutura de decisão e manipulação de dados, utilizando linguagens amplamente utilizadas no mercado.

Além do aprendizado técnico, o projeto incentiva o trabalho em equipe, o planejamento de software e o uso de ferramentas colaborativas, simulando um ambiente real de desenvolvimento. Ao final, os participantes terão construído um jogo funcional, podendo ser executado tanto no navegador (versão web) quanto no terminal (versão em Python).

Linguagens Utilizadas

HTML: Estrutura da interface do jogo na versão web.
CSS: Estilização e design da interface.
JavaScript: Lógica principal do jogo (perguntas, pontuação e interatividade).
Python: Versão alternativa do jogo via terminal, para reforçar conceitos de lógica e estruturas de repetição.
Java: (Opcional) Possível expansão futura com interface gráfica (Swing ou JavaFX).

Ferramentas Utilizadas

VS Code: Editor de código para desenvolvimento.
GitHub: Controle de versão e armazenamento do código-fonte.
Trello: Organização das tarefas e cronograma da equipe.
Figma: Criação do layout e protótipos da interface.
MySQL: (Opcional) Armazenamento de perguntas e pontuações dos jogadores.

Instalação

A seguir estão as instruções para executar o projeto em ambiente local:

Faça o clone do repositório do projeto no GitHub.

Caso utilize a versão web:

Crie uma pasta para o projeto.

Copie os arquivos HTML, CSS e JavaScript para dentro dessa pasta.

Abra o arquivo index.html em um navegador de sua preferência.

Caso utilize a versão em Python:

Certifique-se de ter o Python 3.x instalado.

Execute o arquivo principal com o comando:

python jogo_do_milhao.py
